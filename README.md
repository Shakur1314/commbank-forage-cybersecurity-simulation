A virtual job simulation offered by **Commonwealth Bank** through the Forage platform. This program simulates the responsibilities of a cybersecurity generalist at one of Australia's largest financial institutions — covering fraud data analysis and incident response.

---

## Program Structure

| Task | Title | Tools / Skills |
|------|-------|----------------|
| [Task 1](./task1-splunk-fraud-analysis/README.md) | Splunk Fraud Data Analysis | Splunk Enterprise, SPL, Data Visualisation |
| [Task 2](./task2-incident-response/README.md) | Incident Response | Phishing Analysis, Ransomware Response, IR Lifecycle |

---

## Skills Demonstrated

- SIEM tool operation using Splunk Enterprise
- Fraud data ingestion, SPL querying, and dashboard creation
- Cyber incident identification and attack classification
- Incident containment, recovery, and post-incident reporting
- Structured analyst communication and documentation
